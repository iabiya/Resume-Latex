# ATS-Friendly Resume using LaTeX

This repository contains my professional ATS-friendly resume developed using LaTeX and Visual Studio Code.

## Technologies Used

- LaTeX
- MiKTeX
- Strawberry Perl
- Visual Studio Code
- Git
- GitHub

## Files

- resume.tex – LaTeX source code
- resume.pdf – Compiled resume

## Build Instructions

1. Install MiKTeX
2. Install Strawberry Perl
3. Install Visual Studio Code
4. Install LaTeX Workshop Extension
5. Open resume.tex
6. Build the project to generate resume.pdf
